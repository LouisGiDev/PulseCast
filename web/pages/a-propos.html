Vitrine · Statique

Conçu pour être partagé

Pas de compte, pas de base de données — juste une page qui met tes streamers sous les projecteurs.

PulseCast regroupe les chaînes Twitch de ta communauté sur une seule URL — plus besoin d’éparpiller les liens sur Discord.

Tu choisis les chaînes dans frontend/js/config.js (tableau streamers) : titre affiché + pseudo Twitch. Aucun serveur PulseCast — le lecteur officiel Twitch charge le flux comme sur twitch.tv.

En bref

Partage le lien de ton site : tes membres voient qui est en live, en un coup d’œil. Le bouton « Voir sur Twitch » ouvre la chaîne sur twitch.tv.

Réseaux sociaux

Les icônes en bas de page lisent aussi config.js (section social). Tant qu’une URL est vide, le clic mène vers la page Réseaux pour la configurer.
